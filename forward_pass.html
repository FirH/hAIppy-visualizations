Neural Network Forward Pass — This interactive visualization shows how a small neural network computes a prediction step by step.
Layers: Input x₁ and x₂ are your feature values. Hidden neurons apply weights, add a bias, then pass through an activation function (e.g. ReLU). Output applies a second activation to produce a final prediction.
Controls: Drag x1 / x2 sliders to change inputs. Edit weights in the tables on the right. Hit ▶ Play to step through the forward pass layer by layer, or click the dots to jump to any step. Use ‹ › to step manually. ↺ resets to the static view. speed controls animation pace. font adjusts text size on canvas. H changes canvas height. Randomize generates new weights.
