05 Kernel Trick — 2D → 3D Mapping (RBF) Module 8J
2D Input Space
Not linearly separable
→ φ(x) z = x₁² + x₂²
3D Feature Space
Linearly separable
Class A (inner cluster) Class B (outer ring)
Key insight: The RBF kernel implicitly maps each point x to a higher-dimensional space where classes become separable. Here z = x₁² + x₂² — the squared distance from origin — lifts the inner cluster above the outer ring, making a flat hyperplane sufficient. The kernel trick avoids computing φ(x) explicitly: it only evaluates K(xᵢ, xⱼ) = φ(xᵢ)·φ(xⱼ).
