Activation Functions

A neural network neuron does two things: multiply inputs by weights, then run the result through an activation function. That function decides whether the neuron "fires" — and how loud. Pick one below to explore it.

Why do we need activation functions at all? Without them, stacking layers is just multiplying matrices — the whole network collapses into a single linear equation. Activation functions add non-linearity, which is what lets neural networks learn complex patterns like faces, speech, or handwriting.
